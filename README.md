# Rojman Mobile App

اپلیکیشن اندروید برای نمایش آخرین خبرهای `rojman.org` با امکانات زیر:

- نمایش آخرین خبرها از وردپرس روژمان
- منوی دسته‌بندی با دریافت داینامیک دسته‌ها
- ذخیرهٔ خبرها در بخش «موارد دلخواه» داخل خود اپ
- صفحهٔ اختصاصی فکت‌چک و راستی‌آزمایی با خواندن ۱۰ مطلب آخر از `https://rojman.org/factcheck`
- صفحهٔ ارسال خبر / ارسال مطلب با باز کردن ایمیل به `info@rojman.org`
- نوتیفیکیشن خبر جدید با بررسی دوره‌ای
- لوگوی رسمی روژمان به‌عنوان آیکون اپ

## ساخت خروجی در GitHub Actions

بعد از آپلود پروژه در GitHub:

1. به تب **Actions** بروید.
2. ورک‌فلو **Android Release Build** را اجرا کنید.
3. بعد از پایان اجرا، از بخش **Artifacts** فایل‌های زیر را بگیرید:
   - `rojman-apk`
   - `rojman-aab`

## ساخت محلی

نیازمندی‌ها:

- Android Studio Iguana یا جدیدتر
- JDK 17
- Android SDK 35

دستورها:

```bash
gradle assembleRelease
gradle bundleRelease
```

خروجی‌ها:

- APK: `app/build/outputs/apk/release/`
- AAB: `app/build/outputs/bundle/release/`

## نکته

این پروژه از REST API استاندارد وردپرس (`/wp-json/wp/v2`) برای خبرها و دسته‌بندی‌ها استفاده می‌کند؛ همان قابلیتی که وردپرس به‌صورت رسمی برای خواندن پست‌ها و دسته‌ها ارائه می‌دهد.
